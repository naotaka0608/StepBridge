# STEP to FBX Converter

STEPファイル（.step, .stp）を、3D CGソフト（Unity, Blender, etc.）で利用しやすいFBXやOBJ形式に変換するツールです。
Fusion 360のエクスポート機能と同等の品質（滑らかな法線、正しい色再現、適切な明るさ）を目指して調整されています。

## 特徴
- **高品質メッシュ**: 微細な曲面もカクつかずに滑らかに変換します。
- **正確な色再現**: アセンブリ構造を解析し、部品単位・アセンブリ単位で設定された色（黄色や紫など）を正しく反映します。
- **滑らかな法線**: STEPファイルの曲面情報を利用し、スムーズなシェーディング（スムースシェーディング）を実現しています。
- **明るいマテリアル**: FBX出力時にPhongシェーディングと環境光を適用し、暗くなりすぎないように調整済みです。

## 必要要件
- Windows
- Python 3.10+
- [uv](https://github.com/astral-sh/uv) (高速なパッケージ管理ツール)

## インストール

依存関係をインストールします。

```bash
uv sync
```

## 使い方

以下のコマンドでアプリケーションを起動してください。

```bash
uv run python run_locally.py
```

### 操作方法
1. アプリケーションウィンドウにSTEPファイルをドラッグ＆ドロップします。
2. 出力先のフォルダを指定します（デフォルトは入力ファイルと同じ場所）。
3. 「Convert」ボタンを押すと変換が始まります。
4. 変換が完了すると、同じフォルダに `.fbx` または `.obj` ファイルが生成されます。
